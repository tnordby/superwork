# Superwork Product Overview

## Summary

Superwork is a client platform for a HubSpot consulting business.
It combines client portal functionality with delivery visibility so customers can manage requests, follow projects, communicate with consultants, upload assets, and understand billing and usage in one place.

## What Superwork is

Superwork is:
- a client portal
- a delivery coordination layer
- a communication hub tied to consulting work
- a place for billing and usage transparency
- a system for organizing assets and deliverables

## What Superwork is not

Superwork is not:
- a replacement for HubSpot
- a generic project management tool for every use case
- a fully custom portal per customer
- a public social collaboration product

## Core value

For customers:
- less chaos
- clearer visibility
- easier collaboration with consultants
- fewer invoice and file hunts

For Superwork internally:
- more standardized delivery
- better customer communication
- better auditability
- more scalable operations
