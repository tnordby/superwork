# V1 Scope

## In scope
- authentication with Supabase Auth
- protected routes
- organization-scoped portal access
- customer home dashboard
- projects list and project detail views
- project status and milestone visibility
- contextual messages / feedback tied to projects
- asset upload, storage metadata, download, and deliverable visibility
- billing and invoice visibility
- balance and usage visibility
- team member invitation and membership management
- internal provider workspace for assigned work
- basic admin oversight tools

## Out of scope for v1
- highly customizable per-customer theming
- cross-organization collaboration
- public provider signup
- advanced pricing experimentation
- hour tracking exposed to customers
- complex real-time collaboration beyond what improves core communication
- translation CMS tooling
- customer-facing internal SOP visibility
