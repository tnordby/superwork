# Admin and Employee Workspace

## Provider workspace
Providers need:
- visibility into assigned customers and projects
- the ability to send updates
- the ability to upload deliverables
- a structured workflow for consistent delivery

## Admin workspace
Admins need:
- customer organization management
- membership overrides
- project interventions
- usage adjustments
- audit logs

## Guardrails
- internal views may be denser but should remain consistent with the design system
- internal-only notes and controls must never leak to customers
- all sensitive admin actions should be logged
