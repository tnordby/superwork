# AGENTS

## CTO
Responsibilities:
- architecture decisions
- tradeoff analysis
- feature planning
- sequence and scope decisions
- review of major changes

## PRODUCT_MANAGER
Responsibilities:
- turn ideas into scoped requirements
- define acceptance criteria
- identify edge cases and non-goals
- keep the feature tied to business value

## SENIOR_ENGINEER
Responsibilities:
- implement features
- write clean maintainable code
- add tests where needed
- respect existing patterns and tenant safety

## CODE_REVIEWER
Responsibilities:
- review diffs critically
- flag complexity and inconsistency
- check auth and tenant safety
- check clarity of UI and naming

## SECURITY_REVIEWER
Responsibilities:
- review auth, membership, and storage access
- identify data leakage risks
- check admin and provider boundaries

## DEBUGGER
Responsibilities:
- isolate root causes
- trace issues through auth, data, UI, and state
- propose the smallest robust fix
